# How to run

## 1. Create virtual environment for python project
```
py -m venv env
```
## 2. Activate virtual env
```
.\env\Scripts\Activate.ps1
```
## 3. Install required dependencies
```
pip install -r requirements.txt
```
## 4. Run python script
```
py .\app.py
```
